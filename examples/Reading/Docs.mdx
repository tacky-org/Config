# Reading

Two hooks for reading from a `ConfigLoader`, matching TanStack Query's own naming:

<table>
  <thead><tr><th>Hook</th><th>Use when</th></tr></thead>
  <tbody>
    <tr>
      <td><code>useConfigSuspenseQuery</code></td>
      <td>You want Suspense — wrap with <code>&lt;Suspense&gt;</code> and <code>&lt;ErrorBoundary&gt;</code></td>
    </tr>
    <tr>
      <td><code>useConfigQuery</code></td>
      <td>You want inline loading / error states, no boundaries needed</td>
    </tr>
  </tbody>
</table>

Config is cached indefinitely (`staleTime: Infinity`) — it is loaded once per session
and never re-fetched unless explicitly invalidated.

---

## `useConfigSuspenseQuery` — without context

`data` is always `TRuntime` — the Suspense boundary guarantees the component
only renders after the config has loaded.

```tsx
const suspenseLoader = ConfigLoader.create<AppConfig>({
  key:      'app',
  load:     fromFetch('/api/config'),
  validate: withZod(AppConfigSchema),
});

const { data: config } = useConfigSuspenseQuery(suspenseLoader);
return <p>{config.apiUrl}</p>;
```

<Canvas of={Stories.Suspense} />

## `useConfigQuery` — without context

No `<Suspense>` or `<ErrorBoundary>` needed.

```tsx
const inlineLoader = ConfigLoader.create<AppConfig>({
  key:      'app',
  load:     fromFetch('/api/config'),
  validate: withZod(AppConfigSchema),
});

const { data: config, isLoading, isError, error } = useConfigQuery(inlineLoader);

if (isLoading) return <Spinner />;
if (isError)   return <p>Error: {String(error)}</p>;
return <p>{config!.apiUrl}</p>;
```

## `useConfigSuspenseQuery` — with context

When ctx is provided it is required as the second argument — the type system enforces it.
Each distinct context value gets its own cache entry.

```tsx
interface ThemeCtx { darkMode: boolean }

const themedLoader = ConfigLoader.create<AppConfig, AppConfig, ThemeCtx>({
  key:      (ctx) => ['app', ctx.darkMode ? 'dark' : 'light'],
  load:     (ctx) => fromFetch(`/api/config?theme=${ctx.darkMode ? 'dark' : 'light'}`)(),
  validate: withZod(AppConfigSchema),
});

const { data: config } = useConfigSuspenseQuery(themedLoader, { darkMode: true });
```

## `useConfigQuery` — with context

```tsx
const { data: config, isLoading } = useConfigQuery(themedLoader, { darkMode: false });
```

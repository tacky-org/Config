# Prefetching

Populate the TanStack Query cache before a component tree renders by calling
`queryClient.prefetchQuery` with `loader.queryOptions()`. When `useConfigSuspenseQuery`
then runs, the data is already there — no Suspense fallback, no loading state.

Use `ensureQueryData` instead of `prefetchQuery` when you also need the resolved value back.

---

## `beforeLoad` vs `loader`

<table>
  <thead><tr><th></th><th><code>beforeLoad</code></th><th><code>loader</code></th></tr></thead>
  <tbody>
    <tr><td>Runs</td><td>Before route activates — top-down, in series</td><td>When route is about to render — in parallel with sibling loaders</td></tr>
    <tr><td>Use for</td><td>Auth guards, redirects, context that child routes depend on</td><td>Data fetching — preferred for performance</td></tr>
    <tr><td>Return value</td><td>Not used for data</td><td>Available via <code>Route.useLoaderData()</code></td></tr>
  </tbody>
</table>

## Without context

### No prefetch — Suspense fallback shows

<Canvas of={Stories.Without} />

### Prefetch in `loader` — Suspense fallback never appears

```ts
export const Route = createFileRoute('/app')({
  loader: ({ context: { queryClient } }) =>
    queryClient.prefetchQuery(appConfigLoader.queryOptions()),
});
```

<Canvas of={Stories.Loader} />

### `ensureQueryData` — value available via `useLoaderData`

```ts
export const Route = createFileRoute('/app')({
  loader: async ({ context: { queryClient } }) => {
    const config = await queryClient.ensureQueryData(appConfigLoader.queryOptions());
    return { config };
  },
});
```

<Canvas of={Stories.LoaderData} />

## With context

### No prefetch — each context variant suspends independently

<Canvas of={Stories.WithoutContext} />

### Prefetch all variants in `loader`

Prefetch each context variant upfront so components never suspend.

```ts
export const Route = createFileRoute('/app')({
  loader: ({ context: { queryClient } }) =>
    Promise.all([
      queryClient.prefetchQuery(langConfigLoader.queryOptions({ language: 'en' })),
      queryClient.prefetchQuery(langConfigLoader.queryOptions({ language: 'fr' })),
    ]),
});
```

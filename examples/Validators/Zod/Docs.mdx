# `withZod`

Creates a `validate` function from any schema that exposes `.parse(raw)`.
Works with **Zod** and any compatible library.

```ts
import { z } from 'zod';
import { withZod } from '@tacky-org/config';

const Schema = z.object({
  host:   z.string(),
  port:   z.number(),
  secure: z.boolean(),
});

const loader = ConfigLoader.create({
  load:     fromFetch('/api/config'),
  validate: withZod(Schema),  // type inferred from the schema
});
```

The adapter is duck-typed — it only requires the `.parse(raw)` method, so any
library that matches that shape works without any extra setup.

# `withJoi`

Creates a `validate` function from a Joi schema.

Joi's `.validate()` returns `{ error?, value }` rather than throwing directly.
`withJoi` checks the result and throws the error for you, keeping the behaviour
consistent with the other adapters.

```ts
import Joi from 'joi';
import { withJoi } from '@tacky-org/config';

const Schema = Joi.object({
  host:   Joi.string().required(),
  port:   Joi.number().required(),
  secure: Joi.boolean().required(),
});

const loader = ConfigLoader.create({
  load:     fromFetch('/api/config'),
  validate: withJoi(Schema),
});
```
